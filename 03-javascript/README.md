## 📈 Ожидания по уровням

| Уровень | Ожидания |
|---|---|
| Intern | Понимает базовый синтаксис, типы, массивы, функции и события. |
| Junior | Уверенно работает с async/await, Promise, модулями, массивами/объектами и базовым DOM. |
| Middle | Понимает Event Loop, memory model, edge cases языка, браузерный runtime и умеет отлаживать асинхронность. |
| Senior | Проектирует устойчивые abstractions, избегает runtime-ловушек и оптимизирует критичные участки. |
| Architect | Формирует инженерные стандарты языка, runtime-паттерны и правила проектирования общего слоя логики. |
