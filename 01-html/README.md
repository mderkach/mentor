## 📈 Ожидания по уровням

| Уровень | Ожидания |
|---|---|
| Intern | Знает базовые теги, структуру документа, простые формы и текстовую семантику. |
| Junior | Уверенно верстает страницы с семантикой, медиа, таблицами и формами; понимает lazy loading, srcset, details/summary. |
| Middle | Осознанно использует resource hints, dialog, custom elements basics, SEO/meta, a11y и ограничения нативных элементов. |
| Senior | Проектирует устойчивую систему семантики, форм и SSR/SEO-поведения на уровне продукта. |
| Architect | Определяет стандарты семантики, форм, accessibility и interoperability для всех команд. |
