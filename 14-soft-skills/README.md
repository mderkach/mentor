# 🤝 14. Soft Skills

Этот раздел про поведение инженера в реальной команде: коммуникация, ответственность, ревью, принятие решений и работа с рисками.

---

## 🎯 Что здесь важно

- Это не «общие слова», а навыки, которые напрямую влияют на скорость и качество поставки.
- Работодатель оценивает soft skills каждый день: на стендапах, в PR, в переписке, в спорных решениях.
- Сильные soft skills ускоряют карьерный рост не меньше, чем hard skills.
