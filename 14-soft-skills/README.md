# 🤝 14. Soft Skills

Soft skills во frontend — это не декоративный бонус к hard skills, а способ делать работу прозрачной, предсказуемой и управляемой для команды.

Сейчас домен еще собирается. Публично уже есть полезная база по инженерной коммуникации и ожиданиям по уровням, но полноценный маршрут по темам и практикам еще не собран.

- Если хочешь улучшить ежедневную инженерную работу, начни с коммуникации, ownership и декомпозиции.
- Если тебе нужен ориентир по карьерным ожиданиям, затем переходи к материалу по уровням.
