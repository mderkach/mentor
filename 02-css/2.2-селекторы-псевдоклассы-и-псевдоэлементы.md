# 2.2 Селекторы, псевдоклассы и псевдоэлементы

Селекторы определяют, **кому** применять стиль. Чем аккуратнее они построены, тем стабильнее кодовая база.

---

## 🎯 Что ты освоишь

- как выбирать селекторы без перегруза специфичности;
- где использовать `:hover`, `:focus`, `:focus-visible`, `:disabled`;
- как применять `::before` и `::after` без ломания доступности;
- как читать селектор и сразу понимать риск.

---

## База селекторов

Чаще всего хватает:
- `.class`;
- `element` (осторожно);
- `[attr]` и `[attr="value"]`;
- сочетаний `.block .element` при внятной структуре.

Обычно лучше опираться на классы, а не на глубокую вложенность HTML.

---

## Псевдоклассы состояния

Ключевые для UI:
- `:hover` — наведение;
- `:focus` / `:focus-visible` — фокус клавиатуры;
- `:active` — активное нажатие;
- `:disabled` — недоступный контрол.

Пример:

```css
.button {
  background: #1a73e8;
  color: #fff;
}

.button:hover {
  background: #1557b0;
}

.button:focus-visible {
  outline: 2px solid #0b57d0;
  outline-offset: 2px;
}

.button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
```

---

## Псевдоэлементы

- `::before` и `::after` — декоративные элементы;
- `::marker` — маркер списков;
- `::placeholder` — плейсхолдер формы;
- `::selection` — стиль выделения текста.

Важно:
- не клади в `::before/::after` важный смысловой текст;
- декоративный контент должен оставаться декоративным.

---

## `:is()`, `:where()`, `:not()`

- `:is()` — сокращает повторяющиеся селекторы;
- `:where()` — как `:is()`, но с нулевой специфичностью;
- `:not()` — исключает случаи.

Практика для масштабируемого CSS:
- сложные наборы селекторов выноси в `:is()`;
- низкую специфичность базовых правил держи через `:where()`.

---

## Частые ошибки

- Используют длинные цепочки селекторов вместо понятного класса.
- Ставят стили только на `:hover`, забывая клавиатурный `:focus-visible`.
- Через `::before` вставляют текст, который реально нужен пользователю.
- Лечат конфликт специфичности добавлением ещё более тяжёлого селектора.

---

## 🔧 Практика

1. Перепиши 5 длинных селекторов в более короткие и читаемые.
2. Добавь корректные состояния `hover/focus-visible/disabled` для кнопки и ссылки.
3. Оформи список через `::marker` без лишней вложенности в HTML.
