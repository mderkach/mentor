# 2.6 Фон, границы, эффекты и визуальное оформление

Визуальные стили должны усиливать интерфейс, а не мешать чтению и скорости. Здесь важен баланс: выразительно, но без "декоративного шума".

---

## 🎯 Что ты освоишь

- как работать с фонами, границами и тенями;
- чем `box-shadow` отличается от `filter: drop-shadow()`;
- когда использовать фон в CSS, а когда `img` в HTML;
- как делать аккуратные эффекты без перегруза.

---

## Фоны: однотонный, градиент, изображение

```css
.card {
  background: linear-gradient(180deg, #ffffff 0%, #f5f8ff 100%);
  border: 1px solid #d8e2ff;
  border-radius: 16px;
}
```

Для `background-image` важно задавать:
- `background-size` (`cover`/`contain`);
- `background-position`;
- `background-repeat: no-repeat`.

---

## CSS background vs `<img>`

Используй `background-image`, если изображение **декоративное**.

Используй `<img>`/`<picture>`, если изображение:
- контентное;
- должно иметь `alt`;
- участвует в SEO/доступности.

---

## Границы и радиусы

- `border` задаёт рамку;
- `border-radius` смягчает форму;
- слишком большой радиус на всех компонентах быстро делает UI "игрушечным".

Практика:
- держи 2-3 размера радиусов как системные токены (`8/12/16`).

---

## Тени и эффекты

- `box-shadow` — для обычных блоков;
- `filter: drop-shadow()` — полезно для прозрачных PNG/SVG форм, где нужна тень по реальному контуру.

```css
.card {
  box-shadow: 0 10px 24px rgba(15, 23, 42, 0.12);
}
```

Избегай:
- 4-5 теней сразу;
- слишком размытой "грязи" вместо иерархии слоёв.

---

## Прозрачность

- `opacity` влияет на весь элемент вместе с текстом и детьми;
- `rgba()`/`hsl(... / alpha)` позволяют сделать прозрачным только цвет фона/границы.

---

## Частые ошибки

- Используют CSS-фон для смыслового контента.
- Делают тяжёлые декоративные эффекты без UX-цели.
- Выбирают низкий контраст текста на фоне.
- Не проверяют, как фон/эффекты выглядят на мобильных.

---

## 🔧 Практика

1. Собери 3 варианта карточки: спокойная, акцентная, предупреждение.
2. Реализуй декоративный фоновый паттерн через `background-image`.
3. Сделай один и тот же блок с `box-shadow` и `drop-shadow`, сравни результат.
