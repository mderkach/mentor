## 🎯 Как проходить раздел

1. Выбери 1-2 компетенции под текущие задачи проекта.
2. Закрой ключевые навыки по порядку: понимание -> применение -> проверка.
3. Закрепи результат на практических задачах из [TASKS.md](../TASKS.md).

## 📈 Ожидания по уровням

| Уровень | Ожидания |
|---|---|
| Intern | Понимает селекторы, box model, display, position, базовый flexbox. |
| Junior | Уверенно применяет flexbox/grid, адаптив, units, transitions, typography и базовую архитектуру стилей. |
| Middle | Работает с cascade layers, nesting, custom properties, container queries, subgrid, complex layout и темизацией. |
| Senior | Проектирует design tokens, стратегию каскада, a11y-friendly CSS и performance-sensitive стили. |
| Architect | Определяет стандарты styling, theming, токенов, визуальной консистентности и cross-team CSS governance. |
