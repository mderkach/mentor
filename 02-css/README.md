## Как учиться в этом разделе

- Идём по уровням: **Стажер -> Джун -> Мидл -> Сеньор**.
- В каждом уроке: объяснение, правила выбора инструментов, практика и чеклист.
- Цель: не просто "знаю свойство", а умею сверстать реальный интерфейс без хаоса в стилях.

## 🟣 Сеньор

- Формирует CSS-стандарты команды: токены, слои каскада, качество и критерии ревью.
- Управляет сложностью: чтобы проект рос, а стили не превращались в "важно, но сломано".

## ✅ Ожидания по уровням

| Уровень | Что должен уметь |
|---|---|
| Стажер | Понимает каскад, селекторы, box model, position и уверенно собирает простые layout на flex. |
| Джун | Делает production-вёрстку: grid, типографика, адаптив, эффекты и анимации без ломания UX. |
| Мидл | Осознанно внедряет modern CSS: container queries, layers, nesting, `:has()`, tokens и progressive enhancement. |
| Сеньор | Проектирует стратегию CSS для команды: архитектура слоёв, quality bar, производительность и поддерживаемость. |
