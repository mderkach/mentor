## 📈 Ожидания по уровням

| Уровень | Ожидания |
|---|---|
| Intern | Использует git, package manager, команды запуска и базовые scripts. |
| Junior | Настраивает lint/test/build, понимает env и базовые pipeline шаги. |
| Middle | Работает со сборкой, workspace/monorepo, CI cache, analyzers и package governance. |
| Senior | Проектирует инфраструктуру фронтенда, кэширование, pipelines, observability и supply-chain controls. |
| Architect | Определяет platform engineering стандарты, toolchain strategy и delivery model команды. |
