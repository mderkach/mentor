# ♿ 12. Доступность

Доступность во frontend нужна не как отдельная галочка для compliance, а как базовый quality bar интерфейса: семантика, клавиатура, фокус, screen reader и предсказуемое поведение для реальных пользователей.

Сейчас этот домен пока собран только как каркас. Публично есть один стартовый обзор, но полноценный маршрут по уровням еще не оформлен.

## Статус раздела

- Домен: `⚪ Каркас`
- Уроки: `🟡 Частично готовы`
- Задачи: `⚪ В подготовке`
- Материалы: `⚪ В подготовке`

## С чего начать

- Сначала закрой базу по [HTML](../01-html/README.md) и [CSS](../02-css/README.md), иначе доступность быстро превратится в набор несвязанных правил.
- Используй текущий файл как ориентир по домену, а не как законченный learning path.
