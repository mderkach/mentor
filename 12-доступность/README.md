# ♿ 12. Доступность

Доступность во frontend нужна не как отдельная галочка для формального соответствия требованиям, а как базовое требование к качеству интерфейса: семантика, клавиатура, фокус, screen reader и предсказуемое поведение для реальных пользователей.

Сейчас этот домен пока собран только как каркас. Публично есть один стартовый обзор, но полноценный маршрут по уровням еще не оформлен.
