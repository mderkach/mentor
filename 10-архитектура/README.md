# 🏗 10. Архитектура

Архитектура во frontend нужна не для красивых схем, а для того, чтобы продукт можно было развивать без постоянного распада границ, API и ответственности между слоями.

Сейчас этот домен в состоянии каркаса. Публично есть только один обзорный файл, поэтому раздел пока нельзя считать полноценным маршрутом.
