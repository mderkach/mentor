# 🏗 10. Архитектура

Архитектура во frontend нужна не для красивых схем, а для того, чтобы приложение можно было развивать без постоянного распада границ, API и ответственности между слоями.

> 🚧 🚧 🚧
>
> Раздел все еще в развитии. Часть тем уже можно проходить как учебный мини-трек, но покрытие пока сильнее на `middle/senior`, чем на более ранних уровнях.
>
> 🚧 🚧 🚧

- Не заходи в этот домен слишком рано. Сначала нужна уверенная база по [JavaScript](../03-javascript/README.md), [TypeScript](../05-typescript/README.md), [UI Frameworks](../06-ui-frameworks/README.md) и [Infrastructure](../07-infrastructure/README.md).
