# 🧪 9. Тестирование

Тестирование во frontend нужно не ради галочки в CI, а чтобы код менялся без страха и не ломал продукт в неожиданных местах.

- Если тесты для тебя пока выглядят как случайный набор библиотек, начинай с обзорного слоя.

- До этого домена полезно уверенно пройти [UI Frameworks](../06-ui-frameworks/README.md), [Infrastructure](../07-infrastructure/README.md) и [Static Analysis](../08-static-analysis/README.md).
