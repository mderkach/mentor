# 🚀 13. Performance

Производительность во frontend нужна не ради красивого графика Lighthouse, а чтобы интерфейс быстро грузился, не дергался, не сжирал main thread и не ломал UX под реальной нагрузкой.

Сейчас этот домен пока находится в состоянии каркаса. Публично есть только один обзорный файл, поэтому раздел еще нельзя считать полноценным маршрутом.
